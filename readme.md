## 🆕 Getting Started

- ### **Prerequisites**

  - [Stream Chat](https://getstream.io/)
  - [React](https://reactjs.org/)
  - [Node JS Server](https://nodejs.org/)

<!-- GETTING STARTED -->

- ### **Installation**

  Install: cd /client & /server and run `npm install`!

  Run Project Locally: `npm start` on both server and client
